# FedRAMP OSCAL Baselines

This directory contains the authoritative FedRAMP Rev 5 OSCAL baselines and NIST SP 800-53 Rev 5 catalog.

Run `make baselines` from the project root to download these files from:
- FedRAMP baselines: https://github.com/GSA/fedramp-automation
- NIST catalog: https://github.com/usnistgov/OSCAL

Do NOT manually edit these files. They are the single source of truth for all control data.
